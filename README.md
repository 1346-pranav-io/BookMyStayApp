# BookMyStayApp 🏨

## Project Overview

**BookMyStayApp** is a Java-based Hostel Booking System designed to simulate the process of booking hostel accommodations.
The project demonstrates the basic flow of a Java application, including program execution, JVM invocation, and console-based interaction.

---

## Objective 🎯

To implement an efficient **Hostel Booking System** that allows users to run the application and view the booking system workflow.

---

## Running Procedure ▶️

1. Clone or download the repository.

2. Open the project in **IntelliJ IDEA** or any Java IDE.

3. Locate the file:

   `BookMyStayApp.java`

4. Compile the program.

5. Click the **Run** button or execute using the terminal.

---

## Project Flow 🔄

1. Program starts.
2. JVM invokes the `main()` method.
3. Application name is displayed.
4. Application version is displayed.
5. Program continues to the next use case or exits.
